About AppConfig

AWS AppConfig is a capability of AWS Systems Manager that enables you to create, manage, and deploy application configurations and feature flags.

  • ✓ Manage applications, environments, and configuration profiles
  • ✓ Store hosted configuration versions with versioning support
  • ✓ Create and manage deployment strategies
  • ✓ Start and stop deployments with lifecycle tracking
