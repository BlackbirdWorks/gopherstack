About IoT Data Plane

The IoT Data Plane service provides an HTTP interface for publishing messages to MQTT topics. Messages are delivered to the embedded MQTT broker and can trigger IoT Topic Rules.

  • ✓ Publish messages to any MQTT topic via POST /topics/<topic>
  • ✓ Supports both raw binary payloads and JSON-wrapped payloads
  • ✓ Messages are forwarded to the MQTT broker for IoT Topic Rule dispatch
  • ✓ Compatible with the AWS IoT Data Plane SDK (aws-sdk-go-v2/service/iotdataplane)
