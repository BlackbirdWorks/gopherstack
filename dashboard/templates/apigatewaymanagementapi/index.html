About API Gateway Management API

The API Gateway Management API provides an HTTP interface for sending data to connected WebSocket clients and managing WebSocket connections at runtime.

  • ✓ Send data to connected WebSocket clients via POST /@connections/<connectionId>
  • ✓ Retrieve connection metadata via GET /@connections/<connectionId>
  • ✓ Disconnect clients via DELETE /@connections/<connectionId>
  • ✓ Compatible with AWS SDK v2 apigatewaymanagementapi client
