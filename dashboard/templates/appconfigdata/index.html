About AppConfigData

AWS AppConfigData provides a runtime HTTP API for applications to retrieve deployed configuration data. Applications call StartConfigurationSession to begin a session, then poll GetLatestConfiguration to retrieve current configuration.

  • ✓ Start a retrieval session via POST /configurationsessions
  • ✓ Retrieve configuration via GET /configuration/<token>
  • ✓ Token rotation on each retrieval for safe polling
  • ✓ Compatible with AWS SDK v2 appconfigdata client
